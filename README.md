# 디지털 풍화기

https://juo6442.github.io/weathering-image/

저화질의 이미지를 쉽고 빠르게 만들어 보세요!

## 동작

1. Add noise
    - 이미지 축소
    - 품질을 낮춤
2. Filter & Degrade
    - 이미지 원래 크기로 확대
    - 검정 overlay 레이어 추가
    - 연황색 color-burn 레이어 추가
    - 품질을 낮춤

## 개발

```
개발 서버 시작
npm run start

gh-pages에 배포
npm run deploy
```
